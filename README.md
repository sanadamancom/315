# 五階層方陣 — Fivefold Magic Cube

1〜125の数字を5層×5×5のマスに配置し、**各層の行・列・対角線、層を貫く縦列、そして角から角へ5層を斜めに貫く空間対角線4本の合計をすべて315に揃える**ブラウザパズルです。

🎮 **プレイする:** https://fivefold.sanadaman.com/

## 特徴

- 難易度: ふつう/むずかしい/とても難しい(独立した2マス推理が2〜3回必要な3段階)
- すべての出題は**論理的な推理だけで解けることを生成時に保証**(「ふつう」以上は2マス同時推理が必須。上位ティアほど「1マス推論では埋まらないマス」の下限を引き上げて実難度を担保)
- 自動セーブ: ブラウザを閉じても続きから再開できます
- 難易度ごとのベストタイム記録
- パネルの位置・開閉・ズームも自動保存(画面下の「配置リセット」で初期化可能)
- URL共有: 「今の状態」「問題を最初から」の2種類の共有URLを発行(コンパクトなバイナリ形式)
- お助け機能(確定マスの自動入力)、Undo/Redo (Ctrl+Z / Ctrl+Y)
- キーボード入力対応: マスを選んで数字を打ち、Enterで配置(Escで選択解除)
- 組み合わせ検索: 残りの数字から目標の合計になるペア/トリオを探せます

## 操作方法

| 操作 | 動作 |
|---|---|
| 候補プールの数字をクリック → マスをクリック | 配置 |
| マスをクリック → 数字をクリック/キーボード入力(Enterか別マスクリックで確定) | 配置 |
| 配置済みマスをクリック | プールに戻す |
| 配置済みマスを右クリック | 仮置きマーク切替 |
| Ctrl+Z / Ctrl+Y | 戻す / 進む |
| パネルのヘッダーをドラッグ | パネル移動(📌=位置固定、⬆=最前面固定) |

```
js/main.js        — 状態管理・ユーザー操作・セーブ機能
```
